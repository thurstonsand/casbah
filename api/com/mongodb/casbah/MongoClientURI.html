class MongoClientURI extends AnyRef

Create a new MongoURI with a URI String e.g.:

  • mongodb://localhost
  • mongodb://fred:foobar@localhost/
  • mongodb://server1,server2,server3

See http://docs.mongodb.org/manual/reference/connection-string/
